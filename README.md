# march-am-bookmarklets

## 動作環境

- ES2021
  - Chrome 99 以上

## 利用方法

- [Bookmarklets](https://raw.githubusercontent.com/t-skgm/march-am-bookmarklets/main/dist/bookmarklets.html) を開く
- 使いたい Bookmarklet をブックマークバーにドラッグ

## 開発

```bash
# install pnpm
$ node --version
> v16.11.1
$ corepack enable

# install deps
$ pnpm install

# build scripts
$ pnpm build
```
