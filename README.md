# bookmarklets

## 動作環境

- ES2021
  - Chrome 99 以上

## 利用方法

- [Bookmarklets](https://t-skgm.github.io/bookmarklets/) ページを開く
- 使いたい Bookmarklet をブックマークバーにドラッグ

## 開発

```bash
# install pnpm
$ node --version
> v16.11.1
$ corepack enable

# install deps
$ pnpm install

# build scripts
$ pnpm build
```

ref: [google/closure\-compiler: A JavaScript checker and optimizer\.](https://github.com/google/closure-compiler)
